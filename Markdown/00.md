# 前言

在当今这个数字化时代，互联网已成为连接世界的桥梁，而Web应用作为互联网的核心组成部分，其重要性不言而喻。随着技术的飞速发展和用户需求的日益增长，企业对于能够高效、安全、用户体验优良的Web应用的需求愈发迫切。然而，面对复杂多变的开发环境和不断更新的技术栈，许多旨在培养Web应用开发者的教材，往往过分聚焦于编程语言及前后端框架的技术细节与使用说明，而教学方式又常常陷入应试教育的窠臼，侧重于答题技巧的训练，忽视了针对实际业务需求的项目实践与职场工作指导。这导致不少初学者在完成相关培训课程后，依然感到迷茫，不知如何将所学理论知识有效应用于真实项目开发之中。

为了解决这一问题，卷积传媒重新开发了一套面向前端程序员的培训课程。《Web应用开发项目教材》作为该系列课程中的第二本配套教材，将会致力于通过一系列贴近实战的教学项目，帮助读者系统地掌握JavaScript语言及其相关编程技术在实际生产环境中的运用，解决在前后端开发过程中遇到的实际问题。

## 0.1 内容简介

本教材将采用项目驱动的教学方法，通过五个精心设计的教学项目，逐步引导读者从基础到高级，全面掌握企业级Web应用开发的核心技术和流程。这些项目将各自对应着一个软件工程师在开发企业级Web应用时会可能会被要求实现的业务类型，读者将借由这些项目的具体实践来学习如何使用JavaScript这门计算机编程语言及其相关的第三方应用框架来解决在Web应用的前后端开发过程中会遇到的问题，下面是这些项目的简单介绍。

- **项目1. 将企业官网重构为互联网服务**：在这一章的项目实践中，读者将学会如何将一个传统的企业官方网站重构为可编程的互联网服务，以便该企业日后可以逐步地将该网站升级为一个可为用户提供线上服务的Web应用。在完成本章项目的实践任务之后，读者将会初步了解构建一个企业级Web应用所需执行的基本步骤，以及执行这些步骤所需使用的工具与技术。
- **项目2. 网站用户功能模块的后端实现**：在这一章的项目实践中，读者将学会演示如何为一家企业的官方网站实现其用户功能模块的后端服务，以便网站的前端工程师后续可以这一后端服务来构建相应的Web UI，从而将网站升级成为一个允许用户执行注册、登录等操作的Web应用。在完成本章项目的实践任务之后，读者将会初步了解为企业级网站开发用户功能模块的过程中所需要执行的基本步骤，以及执行这些步骤所需要掌握的技术以及相关的工具。
- **项目3. 用户注册/登录功能的前端实现**：在这一章的项目实践中，读者将学会如何为一家企业的官方网站实现其用户注册/登录功能的前端界面，以便其潜在的客户可以直接基于通用的网页浏览器来获取使用该企业线上服务的用户权限。在完成本章项目的实践任务之后，读者将会初步了解在基于现有的后端服务来构建相应Web UI的过程中所需要执行的基本步骤，以及执行这些步骤所需要掌握的技术。
- **项目4. 用户信息管理功能的前端实现**：在这一章的项目实践中，读者将学会如何为一家企业的官方网站实现其用户信息管理功能的Web UI，以便其用户可以直接通过通用的网页浏览器编辑并提交个人头像、电话号码、收货地址等个人信息，以便更好的使用该企业提供的互联网服务。在完成本章项目的实践任务之后，读者将会初步了解在前端脚本中处理各种不同类型的数据，并将其提交给指定的后端服务，以及执行这些操作所需要掌握的技术。
- **项目5.  用户线上购物功能的前端实现**：在这一章的项目实践中，读者将学会如何为一家企业的官方网站实现其线上购物功能的Web UI，以便其用户可以直接通过网页浏览器来享用其线上点餐服务，以便更好地借助互联网来开展该连锁店的业务。在完成本章项目的实践任务之后，读者将会初步学会如何实现订单管理、线上点餐这两大功能的Web UI，并掌握实现这些功能所需要掌握的技术。

## 0.2 读者须知

由于这是一本专注于介绍如何基于JavaScript这门编程语言以及相关前后端开发技术来完成Web应用开发任务的项目示范类教材，因此，笔者会假设读者已经学会了JavaScript这门编程语言的基本语法，以及使用它进行面向对象编程和异步编程的基础技能。如果读者尚未掌握这些知识，笔者会强烈建议您在开始学习本教材中的内容之前，先去阅读一下本教材附送资料中的《JavaScript快速入门手册》或其他相关领域的书籍，以便您能以更为扎实的基础来应对接下来的项目实践任务。

除了JavaScript语言及其相关框架之外，能使用HTML+CSS这两门标记语言，以及Bootstrap等网页设计框架来完成Web UI的基本设计也是在学习本教材之前，读者必须具备的基础技能。因此，如果读者在学习本教材的过程中，发现自己对网页设计的相关技能还不太熟悉，笔者在这里也会强烈建议您回头去阅读一下本系列教材中的第一本，《HTML5+CSS3网页设计项目教材》或该领域的其他相关书籍，这也将会有助于您开展接下来的项目实践任务。

最后，笔者在这里还需要特别强调一件事：本教材中所有示例项目的实现都是基于各章节中的代码占比及其阅读体验等众多写作因素进行了平衡考虑之后产生的精简版本，它们不会影响到项目在实际生产环境中的正常运作。但是，如果读者想了解这些项目中某些细节问题的解决方案，还需自己去查阅本书附带的源码包。当然，如果想要真正学会并熟练掌握一项计算机技术，最好的办法就是尽可能地在实践中使用它们，在实际项目需求的驱动下模仿、试错并总结使用经验。所以在这里，我个人并不鼓励直接复制/粘贴的本书附带源码包中的代码，我们更期待读者“自己动手”去模仿书中提供的项目，亲手将自己想要执行的代码输入到计算机中，观察它们是如何工作的。然后，试着修改它们，并验证其结果是否符合预期。如果符合预期，就总结当下的经验，如果不符合预期，则去思考应该做哪些调整来令其符合预期。如此周而复始，才能让学习效果事半功倍。

在此，感谢正在阅读的您选择了这本教材，希望它能够帮助您更好地掌握使用JavaScript这门计算机编程语言以及相关的前后端开发技术，并在实际应用中得到提升和应用。我们也欢迎读者提出宝贵的意见和建议，以帮助我们不断改进和完善它。
